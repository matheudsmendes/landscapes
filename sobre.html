Galeria de Artes

O projeto Galeria de Artes é uma atividade educacional proposta pelo Prof. Israel Cunha na disciplina de Tópicos Especiais em SPI, com o objetivo de praticar desenvolvimento web utilizando HTML/CSS e versionamento no GitHub.

Desenvolvido por Matheus Mendes, este espaço foi cuidadosamente criado para compartilhar uma seleção especial de imagens e vídeos que expressam paz, nostalgia e inspiração. Cada obra foi escolhida para transportar o visitante a momentos de serenidade e reflexão.

Esta galeria virtual é uma demonstração do potencial educativo da web para a divulgação artística. Para explorar mais conteúdos e acompanhar novos projetos:

Siga @matheudsmendes no Instagram
